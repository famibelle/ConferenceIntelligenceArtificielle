# Conference Intelligence Artificielle

Notes de conférence sur l'intelligence artificielle.

## Contenu
- Fichier principal : `ConférenceIA.md`
- Objectif : prises de notes, ressources et liens autour de l'IA

## Utilisation
- Ouvrez `ConférenceIA.md` dans votre éditeur pour lire et mettre à jour les notes.
- Aucun build requis.

## Mettre à jour et pousser
```powershell
git add "ConférenceIA.md" README.md
git commit -m "Mise à jour des notes / README"
git push
```

## Contribuer
- Proposez des améliorations via pull requests.
- Gardez un style clair et concis dans les notes.
